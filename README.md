## 核心能力

🚀 **从零开始创作** — 选个题材，AI 帮你想故事概要 → 自动设计角色（含外貌/服装/性格）→ 生成全集大纲 → 逐集编写分镜剧本 → 生成英文 prompt → 出图 → 合成视频。7 步，每步点一个按钮。

📥 **导入已有内容** — 已有小说/剧本？直接粘贴。AI 自动提取角色、整理大纲、改编成漫剧分镜格式。

🔗 **8 大 AI 厂商** — 即梦 Seedream · 可灵 Kling · 智谱 CogView · 通义万相 · MiniMax · DALL-E 3 · Stability AI · 智谱 CogVideoX。用你自己的 API Key，不锁定任何一家。

⚡ **6 个 AI 创作工具** — 续写后续 · 改写剧情 · 爽点优化（一键加反转钩子）· 台词润色（去 AI 味）· 标题生成 · 封面 Prompt

🎬 **视频合成** — 浏览器内合成 MP4（ffmpeg.wasm）+ 导出剪映工程文件

📚 **9 篇知识库** — 剧本创作 · 角色一致性 · 分镜 Prompt · 视频生成 · 配音音效 · 剪辑合成 · 发布备案 · 变现运营

🌐 **零部署** — 打开网页就用，不需要 Docker，不需要服务器

## 7 步全自动工作流

```
① 概要        ② 角色        ③ 大纲        ④ 剧本        ⑤ 分镜        ⑥ 生图        ⑦ 合成
选题材         AI 自动设计    AI 生成全集    AI 逐集编写    AI 生成英文    8 大厂商       浏览器内
AI 帮想故事    3-5 个角色     故事线        分镜剧本       prompt        一键出图       合成 MP4
```

每一步支持 **AI 自动生成** 和 **手动导入/粘贴** 两种方式。所有内容随时可回头编辑。

## 快速开始

### 在线使用（推荐）

直接打开 [ai-manga-workbench.pages.dev](https://ai-manga-workbench.pages.dev)，免费工具无需配置。

### 本地开发

```bash
git clone https://github.com/duolaAmengweb3/manga-ai-factory.git
cd manga-ai-factory
pnpm install
cp .env.example .env.local
# 编辑 .env.local 填入你的 API Key
pnpm dev
```

打开 http://localhost:3000

### 部署到 Cloudflare Pages

```bash
npx @cloudflare/next-on-pages
npx wrangler pages deploy .vercel/output/static
```

## 环境变量

创建 `.env.local`：

```env
MINIMAX_API_KEY=你的AI-API-Key
MINIMAX_BASE_URL=https://api.xiaomimimo.com/v1
MINIMAX_MODEL=mimo-v2.5-pro
ADMIN_SECRET=你的管理员密钥（用于生成激活码）
```

## 技术栈

- **前端**：Next.js 16 + TypeScript + Tailwind CSS v4 + shadcn/ui
- **部署**：Cloudflare Pages + Workers
- **视频**：ffmpeg.wasm（浏览器端，无需服务器）
- **AI**：多厂商适配（OpenAI 兼容格式 + Anthropic + 火山引擎 V4 签名）
- **存储**：localStorage（v1，无需数据库）
